# POS System 
This is the project submission repository. It's a 
POS system fully built to be used in actual real-world 
scenarios such as a grocery store. Where there's inventory 
management. reports exports and cashier system.

# How to install
1. clone the repository
2. enter command `cmake -B build` 
3. enter command `cd build && make`
4. finally `./pos-system`
